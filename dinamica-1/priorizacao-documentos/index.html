    Priorização dos documentos

    Técnica

       A técnica utilizada para priorização de documentos foi o MoSCoW, que é fortemente utilizado para a priorização de requisitos de software, porém aqui adaptado para priorizar artefatos de software.

    A sigla MoSCoW significa:

    • Must Have
    • Should Have
    • Could Have
    • Won’t Have this time
