    Plataforma

    NOME Plataforma
    NOÇÃO Sistema que será desenvolvido.
    CLASSIFICAÇÃO Sujeito
    IMPACTO(S) 1 - O usuário inicia a plataforma;
    2 - O usuário realiza o upload na plataforma;
    3 - O usuário consulta outros projetos na plataforma;
    SINÔNIMO(S) Site, sistema


    Upload

    NOME Upload
    NOÇÃO Ação de submeter projeto dentro da plataforma.
    CLASSIFICAÇÃO Objeto
    IMPACTO(S) 1 - O usuário realiza o upload do projeto;
    2 - O usuário recebe um feedback após um upload;
    3 - O usuário cancela o upload;
    SINÔNIMO(S) Submeter, carregar, enviar.


    Download

    NOME Download
    NOÇÃO Ação de baixar projeto da plataforma.
    CLASSIFICAÇÃO Objeto
    IMPACTO(S) 1 - O usuário realiza o download do projeto;
    2 - O usuário recebe um feedback após um download;
    3 - O usuário cancela o download;
    SINÔNIMO(S) Transferência, descarregamento, cópia, baixar.


    Visitante

    NOME Visitante
    NOÇÃO Pessoa que acessa a plataforma sem estar logado.
    CLASSIFICAÇÃO Sujeito
    IMPACTO(S) 1 - Visitante é cadastrado com sucesso
    SINÔNIMO(S)


    Estudante

    NOME Estudante
    NOÇÃO Pessoa que acessa a plataforma estando logado
    CLASSIFICAÇÃO Sujeito
    IMPACTO(S) 1 - Estudante se loga na aplicação;
    2 - Estudante pode visualizar ou submeter projetos, se cadastrar em disciplina, seguir usuários e realizar comentários
    SINÔNIMO(S) Aluno.


    Professor

    NOME Professor
    NOÇÃO Pessoa que acessa a plataforma estando logado
    CLASSIFICAÇÃO Sujeito
    IMPACTO(S) 1 - Professor se loga na aplicação;
    2 - Professor cria disciplina e escolhe senha para cadastro na disciplina
    3 - Professor pode visualizar ou submeter projetos, se cadastrar em disciplina, seguir usuários, realizar comentários, remover aluno de disciplina e gerenciar disciplina.
    SINÔNIMO(S) Coordenador, Orientador.


    Administrador

    NOME Administrador
    NOÇÃO Pessoa que gerencia a plataforma estando logado
    CLASSIFICAÇÃO Sujeito
    IMPACTO(S) 1 - Administrador gerencia aplicação por meio de permissões
    SINÔNIMO(S)


    Comentar

    NOME Comentar
    NOÇÃO Ação realizada pelo usuário de escrever em uma publicação
    CLASSIFICAÇÃO Objeto
    IMPACTO(S) 1 - O usuário que recebe esta ação de outro usuário é notificado
    2 - O usuário marcado por meio da ação anterior é notificado
    SINÔNIMO(S)


    Seguir

    NOME Seguir
    NOÇÃO Ação realizada pelo usuário quando o usuário deseja ver as publicações de um perfil de forma frequente
    CLASSIFICAÇÃO Objeto
    IMPACTO(S) 1 - Assim que o usuário passar a seguir outro, as publicações deste passam a aparecer em seu feed da aplicação
    SINÔNIMO(S)


    Pesquisar

    NOME Pesquisar
    NOÇÃO Ação realizada pelo usuário quando possui necessidade de procurar por projetos ou outros usuários dentro da plataforma.
    CLASSIFICAÇÃO Objeto
    IMPACTO(S) 1 - Usuário pesquisa pelo nome do usuário ou projeto no campo de pesquisa da plataforma.
    SINÔNIMO(S) Explorar


    Projeto

    NOME Projeto
    NOÇÃO Artefato realizado em uma disciplina pelo estudante.
    CLASSIFICAÇÃO Objeto
    IMPACTO(S) 1 - Usuário realiza upload de projetos.
    2 - O projeto passa a ser visível para outros usuários
    SINÔNIMO(S) Trabalho


    Criar Conta

    NOME Criar Conta
    NOÇÃO Ação realizada pelo usuário para cadastrar uma conta no aplicativo.
    CLASSIFICAÇÃO Objeto
    IMPACTO(S) 1 - Visitante realiza cadastro.
    2 - O Visitante se torna usuário.
    SINÔNIMO(S) Cadastrar


    Realizar login

    NOME Realizar login
    NOÇÃO Ação realizada pelo usuário para acessar uma conta no aplicativo.
    CLASSIFICAÇÃO Objeto
    IMPACTO(S) 1 - Entra como usuário
    2 - Passa a ter posibildiade de realizar funcionalidades de estudantes.
    SINÔNIMO(S) Logar


    Vincular projeto

    NOME Vincular Projeto
    NOÇÃO Ação realizada pelo usuário para colocar um projeto no aplicativo.
    CLASSIFICAÇÃO Objeto
    IMPACTO(S) 1 - Usuário realiza upload de projetos.
    2 - O projeto passa a ser visível para outros usuários
    SINÔNIMO(S) Upar projeto, subir projeto
